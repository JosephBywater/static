Server Favicon BETA

This endpoint allows you to get the favicon of a server. Returns PNG cached for 6 hours.

Warning: Currently we don't support SRV records and some 1.9/1.10 servers, this will be resolved in v4 which is coming soon!

Example Usage

https://#HOST#/v3/server/favicon/mc.hypixel.net

This endpoint is in BETA and may not always respond correctly, use with caution.
