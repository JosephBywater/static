Username to UUID

This endpoint provides the UUID of a given username. Returns JSON or CSV cached for 10 minutes.

Example Usage

https://api.#REGION.mc-api.net/uuid/notch would return:

{
    "full_uuid": "069a79f4-44e9-4726-a5be-fca90e38aaf5",
    "source": "database",
    "query": "notch",
    "name": "Notch",
    "took": 24,
    "uuid": "069a79f444e94726a5befca90e38aaf5"
}

Explanation

Field Type Description
"full_uuid" StringUUID The UUID of the player in Java format
"source" String If the response was from mojang or cached.
"query" String The original query.
"name" String The username in correct case.
"took" Integer How long the query took.
"uuid" String The UUID of the player without hyphens

Multiple Usernames to UUIDs

This endpoint provides the UUIDs of multiple usernames. Returns JSON cached for 10 minutes.

Example Usage

https://api.#REGION#.mc-api.net/uuid/notch,dinnerbone would return:

{
    "failed": [],
    "success": [
        {
            "full_uuid": "069a79f4-44e9-4726-a5be-fca90e38aaf5",
            "source": "cache",
            "query": "notch",
            "name": "Notch",
            "took": 0,
            "uuid": "069a79f444e94726a5befca90e38aaf5"
        },
        {
            "full_uuid": "61699b2e-d327-4a01-9f1e-0ea8c3f06bc6",
            "source": "database",
            "query": "dinnerbone",
            "name": "Dinnerbone",
            "took": 21,
            "uuid": "61699b2ed3274a019f1e0ea8c3f06bc6"
        }
    ]
}

Explanation

You can request upto 5 uuids at a time, any failed requests will be in the failed array.
