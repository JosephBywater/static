UUID to Username

This endpoint provides the username of a given UUID. Returns JSON or CSV cached for 8 hours.

Example Usage

https://#HOST#/v3/name/b05881186e75410db2db4d3066b223f7 would return:

{
    "full_uuid": "b0588118-6e75-410d-b2db-4d3066b223f7",
    "source": "database",
    "query": "b05881186e75410db2db4d3066b223f7",
    "name": "Marc",
    "took": 0,
    "uuid": "b05881186e75410db2db4d3066b223f7"
}

Explanation

Field Type Description
"full_uuid" StringUUID The UUID of the player in Java format
"source" String If the response was from mojang or cached.
"query" String The original query.
"name" String The username in correct case.
"took" Integer How long the query took.
"uuid" String The UUID of the player without hyphens
