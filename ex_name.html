UUID to Player Username example

Example of getting the name for a given UUID:

https://#HOST#/v3/name/b05881186e75410db2db4d3066b223f7

Dashes will be removed from the uuid automatically. The query will return a JSON object:

            {
                "full_uuid": "b0588118-6e75-410d-b2db-4d3066b223f7",
                "source": "database",
                "query": "marc",
                "name": "Marc",
                "took": 0,
                "uuid": "b05881186e75410db2db4d3066b223f7"
            }


This object conatins the case-correct username associated with the query UUID. If the UUID is not registed with mojang, the following will be returned:
{"error":"Unknown UUID","query":"unknown"}

All queries that go to mojang will take slightly longer than others.

Field Object Description
"full_uuid"StringUUIDThe uuid of the player in java format.
"source"StringIf the server looked in the database, cache or mojang.
"query"StringThe original query.
"name"StringThe username in correct case.
"took"IntegerHow long the query took.
"uuid"StringThe uuid without "-"'s
