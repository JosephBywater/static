Username History

This endpoint provides a list of usernames a UUID has been known as in the past. Returns JSON cached for 15 minutes.

Example Usage

https://#HOST#/v3/history/61699b2ed3274a019f1e0ea8c3f06bc6 would return:

{
    "history": [
        {
            "name": "Dinnerbone"
        }
    ],
    "source": "cache",
    "query": "61699b2ed3274a019f1e0ea8c3f06bc6",
    "took": 0
}

Explanation

Field Type Description
"history" Array A list of usernames this UUID has been associated with in the past. If a changedToAt field is not present then this is the current username.
"source" String If the response was from mojang or cached.
"query" String The original query.
"took" Integer How long the query took.
