UUID to Profile Properties

This endpoint provides the Profile Session Properties of a given uuid. Returns JSON or CSV cached for 15 minutes.

Example Usage

https://api.#REGION#.mc-api.net/properties/a8889e3068b84ce8963f4ea259c3ebe3 would return:

{
    "query":"a8889e3068b84ce8963f4ea259c3ebe3",
    "source":"mojang",
    "properties":{
        "name":"textures",
        "value":"eyJ0aW1lc3RhbXAiOjE0NjMzNzc1Nzg4NDksInByb2ZpbGVJZCI6ImE4ODg5ZTMwNjhiODRjZTg5NjNmNGVhMjU5YzNlYmUzIiwicHJvZmlsZU5hbWUiOiJuamJfc2FpZCIsInRleHR1cmVzIjp7IlNLSU4iOnsidXJsIjoiaHR0cDovL3RleHR1cmVzLm1pbmVjcmFmdC5uZXQvdGV4dHVyZS9hZWU5ZDI0MzgyYWRmOGMyNjQ5ODEyYTliNDZkNTc3M2VhZWQxMzIzN2ZjYTk5MmJhOGZlODVjNjkzZDk4MTEifX19"
    },
    "properties_decoded":{
        "timestamp":1463377578849,
        "profileId":"a8889e3068b84ce8963f4ea259c3ebe3",
        "profileName":"njb_said",
        "textures":{
            "SKIN":{
                "url":"http://textures.minecraft.net/texture/aee9d24382adf8c2649812a9b46d5773eaed13237fca992ba8fe85c693d9811"
            }
        }
    },
    "took":131
}

Explanation

Field Type Description
"query" String The original query.
"source" String If the response was from mojang or cached.
"properties" JsonObject The properties field from mojang.
"properties_decoded" JsonObject The properties 'value' decoded from base64.
"took" Integer How long the query took.
