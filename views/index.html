Usage

Our APIs are simple to use, just follow the instructions below:

  1. Find the API you want to use.
  2. Register for an API key (why do I need this?)
  3. Send your request with API key (Show me an example!)
{{#mcapi.soft_rate_limit}}

Can I use the API without an API key?

Yes, temporarily we are allowing limited access without an API key. This is to allow people to transition and update their applications to use an API key.

{{/mcapi.soft_rate_limit}}

Do all APIs require an API key?

No, the server ping and server favicon APIs do not require an API key.

Also if we have something cached we will not check or require an API key to retrieve it.
