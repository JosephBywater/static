Server Info

This endpoint allows you to get information about a server such as player count, motd and favicon. Returns JSON or CSV cached for 5 minutes.

Example Usage

https://#HOST#/v3/server/info/hivemc.eu would return:

{
    "source": "ping",
    "favicon": "https://#HOST#/v3/server/favicon/hivemc.eu",
    "status": true,
    "players": {
        "max": 20000,
        "online": 9056
    },
    "motd": "The motd for hivemc.eu",
    "version": {
        "protocol": 4,
        "name": "BeeCord 1.8"
    }
}

Explanation

Field Type Description
"favicon" URL URL to another endpoint to retrieve the favicon
"status" Boolean If the server is online or offline
"players" Nested JSON Object Data about player count
"motd" String The server MOTD at time of ping.
"version" Nested JSON Object Data about server version
"source" String If the response was directly from the server or cached.
