Migration Status

This endpoint allows you to check if an account is migrated or not. Returns JSON cached for 8 hours.

Example Usage

https://#HOST#/v3/migrated/Marc would return:

{
    "source": "mojang",
    "query": "Marc",
    "took": 116,
    "migrated": true
}

Explanation

Field Type Description
"migrated" Boolean If the account is migrated or not
"source" String If the response was from mojang or cached.
"query" String The original query.
"took" Integer How long the query took.
