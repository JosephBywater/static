Hello

This is mc-api.net, a collection of fast and free Minecraft related APIs.

What APIs do we offer?

There are a few APIs you can use:

Function Description Path Valid response types Example
Migration Status Check if an account is migrated /v3/migrated JSON /example/migrated
Server Info Get the info about a server /v3/server/info JSON or CSV (Plain Text) /example/ping
Server Favicon Get the Favicon of a server /v3/server/favicon PNG /example/favicon
Server Blacklist Check Check if a server is blacklisted by Mojang /v3/server/blacklist JSON Coming in v4!

By default, endpoints which offer JSON responses will have documentation in JSON. To get a response in CSV (Plain Text) append your request with /csv

Eg: https://#HOST#/v3/server/info/hivemc.eu/csv would give you information about TheHive in CSV format.

A couple of rules

There are just a couple of things that aren't allowed such as:

  • Attempting to "flood" the service
  • Using the service for any mailicious or illegal acions
  • Attempting to bypass any safeguards in place

You acknowledge by using the service that all requests are logged and you may be denied access to the service at any time without prior notification or approval. However a reason may be provided.
